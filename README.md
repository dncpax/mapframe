# mapframe

Objetivo é ter um cliente webmap que possa ser usado no appsmith sem outro middleware.

v0.1 - geojson em var fixa adicionado ao mapa - problemas potenciais: i) limite de dados com performance; ii) appsmith pode ter limites muito inferiores ao browser e começar a dar problemas com poucos registos;
v0.2 - pontos e polys funcionais, boa perf. tudo num único layer.
